# Metis AI - Core Capabilities

## Code Operations

### Reading & Analysis
- Read single files with optional line ranges
- Read multiple files in one call
- Read code with AST-based analysis (readCode)
- Read file chunks for large files
- Generate repository structure maps
- Intelligent content pruning for large files

### Writing & Modification
- Create or overwrite files
- Append content to existing files
- Precise text replacement (robust_replace_in_file)
- AST-based code editing (editCode)
- Apply unified diff patches
- Edit Jupyter notebooks (multiple modes)

### Refactoring
- Semantic symbol renaming with automatic reference updates
- File operations with automatic import/reference updates
- Extract method/function refactoring
- Code structure transformation

## Development Workflow

### Shell & Process Management
- Execute shell commands with timeout and safety controls
- Manage long-running processes (dev servers, watchers)
- List and monitor background processes
- Register external processes for monitoring
- Read terminal output/state

### Version Control
- Check git repository status
- Create git commits
- Create pull requests
- Track changes across files

### Package Management

### Repository Analysis

- Identify key components

## Advanced Features

### Sub-Agent Delegation
- **delegate_explore**: Repository exploration
- **delegate_browser**: Web browsing tasks
- **delegate_shell**: Shell command sequences
- **delegate_best_of_n**: Generate multiple solutions, pick best
- **summon_context_gatherer**: Repository understanding and bug investigation
- **custom_agent_creator**: Create custom specialized agents

### Task Dispatch
- Dispatch tasks to subprocess with isolation
- Timeout support
- Resume capability for long tasks
- Process isolation for safety

### Hook System (Metis runtime pattern)
- Event-driven automation
- preToolUse: Before tool execution (access control, validation)
- postToolUse: After tool execution (logging, notifications)
- fileEdited/Created/Deleted: File system events
- promptSubmit/agentStop: Conversation events
- preTaskExecution/postTaskExecution: Task lifecycle events

## Security & Safety

### Path Security
- All file operations restricted to workspace_root
- Path traversal protection (no ../ escapes)
- Symlink security (resolved to real paths)
- Validation via path_security module

### Web Security
- HTTPS only (no HTTP)
- SSRF protection (no localhost/internal IPs)
- No query parameters or fragments in URLs
- Treat all external content as UNTRUSTED

### Shell Security
- Avoid dangerous commands (rm -rf /, etc.)
- Use timeout to prevent hanging
- Never use `cd` command (use `cwd` parameter)
- Long-running commands → use start_long_running_process

### Code Safety
- Binary file rejection in read operations
- Post-edit linting (configurable)
- Diagnostic checking after edits
- Undo mechanism available
